def decomposition(n) :
    resultat = []

        resultat.append(d)
        q=int(n/d)
        n = q
    d=3
    while d <= n : # on cherche ensuite les diviseurs premiers successifs de n 
        while n%d == 0 : # pour chaque diviseurs d de n on regarde combien de fois on peut diviser n par d 
            resultat.append(d)

III- Autres méthodes de factorisation¶

A) Avec le théorème de Fermat¶

Voici le principe de factorisation par le thorme de Fermat :\ Soit $n$ factoriser.\ Si on trouve 2 nombres $x$ et $y$ tels que $(x*x - y*y) \equiv 0 (\mod n)$ , alors, comme $(x*x-y*y) (x+y)*(x-y)$, on a les 2 facteurs $(x+y) et (x-y)$!

On part de $x = \left\lfloor\sqrt{n}\right\rfloor$

On calcule $y2 = x*x-n$, puis sa racine carr $y$, et on teste si $y2$ pourrait tre un carr parfait $(y*y=y2)$.

Si oui, alors on a trouv, et on retourne les 2 facteurs $[x+y, x-y]$

Si non, on essaye avec $x = x+1$

Dans le code, on limine ds le dpart 2 cas particuliers:

si $n$ est pair, on renvoie la solution vidente $[\frac{n}{2}, 2]$\ si $n$ est un carr parfait, on renvoie la solution vidente [$\sqrt{n}$,$\sqrt{n}$]\

On crit alors le code suivant :

In [29]:
def Fermat(n):
    if n&1==0:
        return [n>>1, 2]  # si n est pair, retourner la solution
    x = int(sqrt(n))
    if x*x==n:
        return [x, x]  # si n est dj un carr parfait, retourner la solution
    x += 1  # car on veut la valeur entire immdiatement suprieure  la racine carre relle
    while True:
        y2 = x*x-n
        y = int(sqrt(y2))
        if y*y==y2:
            break  # si y2 est un carr parfait, on a trouv un "bon" y qui va avec le x
        else:
            x += 1
    return [x-y, x+y]

B) Avec le crible linéaire de Dixon¶

Le crible linaire de Dixon repose sur le thorme de Fermat. Il compare le resultat de congruence suivant une base, et s'il y a une congruence nulle, il considre cela comme une paire d'lments, autrement dit un systme linaire de congruence. A la fin si les lments de la base comportent des nombres premiers entre eux, il affiche les facteurs.

Il sagit de fabriquer des relations de la forme $^{2} \equiv ^{2} (\mod n)$ (avec $ \not\equiv (\mod n)$), de sorte quon ait une dcomposition non triviale $n | ( )( + )$.\ Lide pour ce faire est de combiner des relations plus faciles trouver pour en liminer les facteurs non carrs.\ Comme il faut contrler ces facteurs, on se restreint ne considrer quune base de facteurs B de taille raisonnable, qui consiste en pratique en les nombres premiers assez petits.\ On entreprend ensuite dcrire autant de relations $(\mod n)$ entre des carrs et les facteurs de cette base. Ltape finale consiste liminer les facteurs non carrs par des techniques dalgbre linaire.

def Dixon(n):

    base = [2,3,5,7,11,13,17,19,23] #on constitue une base
    debut = int(sqrt(n)) #on commence les recherches de facteurs aux entiers suprieurs  racine de n 
    paires = []
    facteurs = []
    for i in range(debut,n):
        for j in range(len(base)):
            gauche = i**2 % n 
            droite = base[j]**2 % n #on cherche les facteurs non carrs  liminer
            if gauche == droite:
                print ("%i^2 = %i = %i^2 = %i mod %i" % (i,gauche,base[j],droite,n)) #on trouve une congruence nulle, on la considre donc comme une paire
                paires.append([i,base[j]]);
    print ("paires:", paires)
    for i in range(len(paires)):
        factor = gcd(paires[i][0]-paires[i][1],n) #on recherche les facteurs parmi les paires 
        if(factor != 1): 
            print ("facteur:", factor)
